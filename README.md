# Оно тебе надо — аукцион вещей, в которые никто не верил

Одностраничный сайт для необычного аукциона, где представлены лоты, которые когда-то не были оценены по достоинству, но сейчас стали культовыми.

## Описание проекта

Сайт представляет собой лендинг, рассказывающий о концепции аукциона. Здесь собраны нестандартные лоты: фильм режиссёра-самоучки, книга, описывающая один день, и абстрактная картина. Проект демонстрирует, как непризнанные шедевры обретают ценность со временем.

## Функциональность

- Адаптивная вёрстка с использованием гридов и флексов
- Семантическая структура HTML
- Стилизация в современном минималистичном стиле
- Интерактивные элементы (кнопки, ссылки)
- Кастомные шрифты Raleway

## Технологии

- HTML5
- CSS3
- Flexbox
- Grid Layout
- Кастомные шрифты (woff, woof2)

## Секции сайта

1. **Шапка (Header)** — навигация по сайту, логотип, контактная информация
2. **Cover (Обложка)** — главный заголовок и призыв к действию
3. **Lots (Лоты)** — карточки с описанием лотов аукциона
4. **About (Об аукционе)** — описание концепции проекта
5. **Подвал (Footer)** — контакты, меню, ссылки на соцсети

## Особенности вёрстки

- Использование `grid-template-columns: 1fr auto 1fr` для симметричного расположения элементов в шапке и подвале
- Оверлей (полупрозрачный слой) для затемнения фоновых изображений
- Адаптивные карточки лотов с `flex: 1 1 334px`
- Круглый логотип в секции "Об аукционе"
- Кастомные маркеры списков
